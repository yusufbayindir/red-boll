# Retention Report - Mastery Badge Loop

## Source

Agent: Mencius  
Task: `TASK-0004`  
Date: 2026-05-11

## Position

Red Ball should not compete as another red-ball platformer with hearts. The retention opportunity is mastery, not scarcity.

## Differentiation Directions

- Momentum mastery: readable clean rolling lines, bounce arcs, stomp timing, and route flow.
- Risk routes: safe path plus optional skill routes per level.
- Stunt objectives: `No Damage`, `All Coins`, `2 Enemy Stomps`, `Perfect Bounce Route`, `Fast Clear`.
- Daily micro course: useful later, not MVP.
- Expressive toy-world identity: tactile physics, bounce sounds, roll feedback, playful dressing.

## Recommended MVP Loop

Build the Mastery Badge Loop first:

- `Clear` unlocks next level.
- `All Coins` rewards route discovery.
- `Clean Run` rewards skill without requiring a harsh timer.
- Level select shows badge progress per level.
- Completion screen shows what improved and what remains.

## Rejections

- Hearts as main retention: hostile in a physics platformer where failure is how players learn.
- Generic skins/worlds/more levels as primary hook: not enough identity.
- Upgrade trees, currencies, gacha cosmetics, daily streak pressure, ad-driven continues: too heavy before replay value is proven.

## PM Note

This aligns with the Trickline proposal but suggests a safer MVP: mastery badges before more demanding route contracts, timers, or ghosts.
