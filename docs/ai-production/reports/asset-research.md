## License Cautions

- Prefer CC0 for first import because it avoids attribution and copyleft surprises.
- Still keep a local `THIRD_PARTY_ASSETS.md` entry for every imported asset with source URL, creator, license, download date, and original zip/license file.
- For CC-BY assets, attribution must be available in-game or in app/store credits. Game-icons.net is usable only if icon-level credits are tracked.
- Avoid CC-BY-NC and "personal use only" assets for this project because ads, IAP, and store distribution are commercial.
- For OpenGameArt collections, do not trust only a collection-level credits file; verify each selected child asset and its license page.
- For itch.io free/name-your-own-price packs, capture the asset page URL and license text at approval time because creators can later edit pages.
- For Google Fonts/OFL, include the OFL license and copyright metadata when distributing font files. Do not sell font files by themselves, and rename modified fonts if required by reserved font names.
- For MIT/ISC/Apache icon libraries, include required notices if source/package files ship with the game or tooling. Do not treat brand/trademark icons as free branding rights.

## Recommendation

Start with a CC0 stack: Kenney UI Pack, Kenney Game Icons, Kenney Particle Pack, Kenney UI/Interface/Impact Audio, Kenney New Platformer Pack, plus one art-direction pass using Pixel Frog Pixel Adventure 1 or Ansimuz Sunny Land as style references. Use Fredoka or Nunito for early UI typography. Keep Game-icons.net and Freesound as controlled fallbacks only after the credits pipeline exists.
