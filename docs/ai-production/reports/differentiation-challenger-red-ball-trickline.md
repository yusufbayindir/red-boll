# Differentiation Challenger Report - Red Ball: Trickline

## Source

Agent: Erdos  
Task: `TASK-0004`  
Date: 2026-05-11

## Strongest Hook

Make the game `Red Ball: Trickline`: compact physics stunt-puzzle levels where the player is not only reaching the flag, but chasing a visible route contract.

Possible contracts:

- Collect coins in sequence.
- Keep momentum.
- Hit bounce pads in a chain.
- Take no damage.
- Finish under par time.
- Clear a perfect line.

## Recommended MVP Loop

Level select -> choose unlocked stunt level -> finish to unlock next -> replay to complete Trickline Contract -> earn medal -> improve ghost/time -> optional daily contract.

## Strongest Arguments

- Current mechanics already support this direction: coins, bounce pads, hazards, enemies, moving platforms, saves, level select.
- It creates replay value without needing dozens of new levels immediately.
- It differentiates from generic red-ball platformers through mastery and route design.
- Hearts should not be the retention core; mastery should.

## Rejected Weaker Hooks

- More levels only: adds volume but not identity.
- Skins only: rewards attachment but does not create attachment.
- Harder levels only: risks frustration without a clear mastery fantasy.
- Hearts as main retention: risks feeling hostile.

## Risks

- Contracts will expose weak level design if levels stay flat.
- Physics must be consistent enough for mastery play.
- Daily challenge should wait until base contracts feel good.

## PM Note

This is a strong candidate direction. Do not implement yet. PM will compare against Game Design / Retention and Level Designer outputs before opening implementation tasks.
