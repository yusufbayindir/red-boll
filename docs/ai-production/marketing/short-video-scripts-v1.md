# Short Video Scripts V1 - Red Ball

## Source

Owner: Marketing Production  
Date: 2026-05-12  
Inputs: `marketing-launch-research.md` video hooks plus current PM decisions.

## Production Rules

- Format: vertical 9:16.
- Length: 7-12 seconds each.
- Use real gameplay only.
- First frame should already be gameplay. No logo intro.
- Keep overlays short enough to read on mute.
- Do not show unimplemented bosses, skins, badges, daily challenges, leaderboards, ghosts, or fake store UI.
- End on a clean finish, a funny failure, or a retry decision. Do not end on a static menu unless the script says so.

## Scripts

### 1. Can You Land This Bounce?

Length: 8 seconds  
Goal: show physics, bounce pads, coins, and danger in one clip.

| Time | Visual | Overlay | Audio/Caption |
| --- | --- | --- | --- |
| 0.0-1.0 | Ball rolling toward bounce pad with hazard visible. | `Can you land this bounce?` | Bounce sound should hit quickly. |
| 1.0-3.5 | Ball hits bounce pad and arcs over danger. | | Keep camera centered on landing. |
| 3.5-6.5 | Ball clips through or near a coin line and lands safely. | `Clean line.` | |
| 6.5-8.0 | Continue rolling toward finish or next platform. | `Try the run` | Caption: `One jump changes the whole route.` |

Capture note: choose a bounce where the landing is visible before the jump starts.

### 2. The Safe Path Is Slower

Length: 10 seconds  
Goal: make route choice and replay value visible.

| Time | Visual | Overlay | Audio/Caption |
| --- | --- | --- | --- |
| 0.0-1.5 | Ball pauses near split route or obvious safe platform. | `The safe path is slower.` | |
| 1.5-4.0 | Quick cut: safe path goes around coins. | `Safe...` | |
| 4.0-8.0 | Replay or second take: player jumps into riskier coin line. | `...or clean?` | |
| 8.0-10.0 | Ball lands and keeps momentum. | `Find the line` | Caption: `Would you take the coins?` |

Capture note: if no clean route split exists, use a coin line that visibly asks for a harder jump.

### 3. One Platform. Bad Timing Hurts.

Length: 9 seconds  
Goal: sell moving-platform timing.

| Time | Visual | Overlay | Audio/Caption |
| --- | --- | --- | --- |
| 0.0-1.2 | Moving platform crosses over hazard. | `One platform.` | |
| 1.2-3.0 | Ball starts too early or barely waits. | `Bad timing hurts.` | |
| 3.0-6.5 | Near miss or clean landing on the platform. | | Let the tension breathe. |
| 6.5-9.0 | Ball jumps off platform safely. | `Time every move` | Caption: `Wait for it... then roll.` |

Capture note: pick a frame where the platform path is readable.

### 4. I Missed One Coin

Length: 11 seconds  
Goal: turn coin collection into an emotional retry hook.

| Time | Visual | Overlay | Audio/Caption |
| --- | --- | --- | --- |
| 0.0-2.0 | Ball reaches a finish path with one visible missed coin behind or above. | `I missed one coin.` | |
| 2.0-4.0 | Hard cut back to the jump before the missed coin. | `Run it back.` | |
| 4.0-8.0 | Player takes the harder jump and grabs the coin. | | Coin sound should be audible. |
| 8.0-11.0 | Ball lands and continues. | `Worth it.` | Caption: `The risky coin line always wins.` |

Capture note: avoid implying a badge reward. This is about the player's own completion urge.

Length: 8 seconds

| Time | Visual | Overlay | Audio/Caption |

Capture note: use the clearest enemy silhouette and avoid overlapping UI.

### 6. This Jump Looks Too Early

Length: 9 seconds  
Goal: create comment bait around timing.

| Time | Visual | Overlay | Audio/Caption |
| --- | --- | --- | --- |
| 0.0-1.5 | Ball approaches a gap before platform feels ready. | `This jump looks too early.` | |
| 1.5-4.0 | Player jumps before the viewer expects it. | | |
| 4.0-6.5 | Moving platform arrives under the ball. | `Wait for it...` | |
| 6.5-9.0 | Ball lands and keeps momentum. | `Clean.` | Caption: `Too early or perfect timing?` |

Capture note: strongest if the landing feels surprising but fair.

### 7. No Brakes, Just Line

Length: 10 seconds  
Goal: sell momentum without promising future Trickline contracts.

| Time | Visual | Overlay | Audio/Caption |
| --- | --- | --- | --- |
| 0.0-1.0 | Ball already rolling. | `No brakes.` | |
| 1.0-4.0 | Jump, bounce pad, or platform chain with no pause. | | Keep cuts minimal. |
| 4.0-7.5 | Ball collects a coin line while staying smooth. | `Just line.` | |
| 7.5-10.0 | Safe landing or finish approach. | `Roll cleaner` | Caption: `One smooth route feels better.` |

Capture note: this should be one continuous take if possible.

### 8. The Level Is Easy Until Here

Length: 10 seconds  
Goal: honestly frame difficulty and show escalation.

| Time | Visual | Overlay | Audio/Caption |
| --- | --- | --- | --- |
| 0.0-2.0 | Calm opening platform and easy jump. | `The level is easy...` | |
| 2.0-4.0 | Camera reaches a tighter hazard or moving platform section. | `...until here.` | |
| 4.0-8.0 | Player clears or narrowly fails the section. | | Use the more dramatic take. |
| 8.0-10.0 | Cut to instant retry or clean recovery. | `One more try` | Caption: `Readable, but not free.` |

Capture note: a clean fail is usable if it clearly invites a retry.

### 9. Thumb Controls Check

Length: 8 seconds  
Goal: reduce mobile-control anxiety.

| Time | Visual | Overlay | Audio/Caption |
| --- | --- | --- | --- |
| 0.0-1.5 | Gameplay with left/right control and jump button visible. | `Thumb controls check.` | |
| 1.5-4.0 | Player rolls, jumps, and corrects mid-run. | `Roll + jump` | |
| 4.0-6.5 | Ball clears a simple hazard. | | |
| 6.5-8.0 | Continue to coin or finish. | `Quick runs` | Caption: `Simple controls, real timing.` |

Capture note: show controls only if they are visually polished and do not cover the obstacle.

### 10. Clean Run Or Reset?

Length: 12 seconds  
Goal: tee up future mastery positioning without claiming badges.

| Time | Visual | Overlay | Audio/Caption |
| --- | --- | --- | --- |
| 0.0-1.5 | Ball starts a smooth section. | `Clean run?` | |
| 1.5-5.0 | Player clears several beats cleanly. | | Let momentum sell the clip. |
| 5.0-7.0 | One tiny mistake: missed coin, clipped hazard, or awkward landing. | `Or reset?` | |
| 7.0-10.0 | Instant retry starts from earlier point. | `Again.` | |
| 10.0-12.0 | Better attempt begins cleanly. | `Find the line` | Caption: `That one mistake always matters.` |

Capture note: do not show a Clean Run badge unless the feature ships. The phrase here is an informal player challenge only.
